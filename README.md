# 🏚️🍌 AI Home Renovation Planner Agent

### 🎓 Step-by-Step Tutorial  
👉 https://www.theunwindai.com/p/build-an-ai-home-renovation-planner-agent-using-nano-banana  

A powerful **multi-agent AI system** built using Google ADK that analyzes room images, creates personalized renovation plans, and generates photorealistic designs using advanced multimodal AI models.

---

## 🚀 Features

- 🔍 **Smart Image Analysis**  
  Upload room & inspiration images — AI understands layout, condition, and style.

- 🎨 **Photorealistic Rendering**  
  Generates realistic renovation visuals using AI.

- 💰 **Budget-Aware Planning**  
  Tailored suggestions based on your budget.

- 📊 **Complete Renovation Roadmap**  
  Timeline, cost breakdown, materials, and contractor guidance.

- 🤖 **Multi-Agent Architecture**  
  Uses Coordinator + Sequential Pipeline pattern.

- ✏️ **Iterative Editing**  
  Refine designs based on your feedback.

---

## 🧠 How It Works

This system uses a **multi-agent pipeline**:

### 1. 📸 Visual Assessor
- Analyzes uploaded room images  
- Extracts style from inspiration images  
- Identifies improvement opportunities  

### 2. 🎨 Design Planner
- Creates budget-based renovation plans  
- Suggests materials, colors, and layouts  

### 3. 🏗️ Project Coordinator
- Generates renovation roadmap  
- Creates photorealistic renderings  
- Provides timeline + cost breakdown
